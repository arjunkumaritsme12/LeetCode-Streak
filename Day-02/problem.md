# Plus One

Link: https://leetcode.com/problems/n-repeated-element-in-size-2n-array/?envType=daily-question&envId=2026-01-02

Approach: 
Humein array me wo number dhoondhna hai jo baar-baar repeat hota hai.

Hum har element ko uske next aur next-to-next element se compare karte hain.

Agar dono me se kisi ke saath match mil jaye, wahi number answer hota hai.

Agar loop me na mile, to last element hi repeated number hota hai.
