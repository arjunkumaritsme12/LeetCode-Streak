# Plus One

Link: https://leetcode.com/problems/trionic-array-i/description/?envType=daily-question&envId=2026-02-03

Approach: 

➡️ Traverse the array once using a single pointer

➡️ First, check for a strictly increasing sequence

➡️ Then, check for a strictly decreasing sequence

➡️ Finally, check again for a strictly increasing sequence

➡️ Ensure all three parts exist and cover the full array
