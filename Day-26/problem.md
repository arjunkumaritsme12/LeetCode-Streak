# Plus One

Link: https://leetcode.com/problems/minimum-absolute-difference/description/?envType=daily-question&envId=2026-01-26

Approach: 

➡️ Sort the array

➡️ Keep a variable to store minimum difference

➡️ Check difference of adjacent elements

➡️ If a smaller difference is found → update minimum & clear result

➡️ If difference equals minimum → add the pair

➡️ Return all collected pairs
