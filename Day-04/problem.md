# Plus One

Link: https://leetcode.com/problems/four-divisors/description/?envType=daily-question&envId=2026-01-04

Approach: 

➡️ Loop up to the square root of each number 
➡️ Count only two non-trivial divisors 
➡️ Check conditions precisely so that the number has exactly 4 divisors 
➡️ Sum them if valid
